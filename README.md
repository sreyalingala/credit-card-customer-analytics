# Credit Card Customer Behavior Analytics

This project analyzes credit card customer behavior using Python for data analysis and Tableau for visualization.  
The goal is to identify spending patterns, customer segments, and churn risk through exploratory data analysis and interactive dashboards.

## Tools Used
Python (Pandas, NumPy, Seaborn)
Tableau
Jupyter Notebook

## Project Structure

data/ — raw and cleaned datasets  
notebooks/ — Python analysis notebooks  
tableau/ — Tableau dashboards  
images/ — dashboard screenshots  

## Key Objectives

• Analyze customer transaction behavior  
• Identify churn patterns  
• Segment customers by spending and usage  
• Visualize insights through dashboards
