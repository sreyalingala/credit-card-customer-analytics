# Credit Card Customer Behavior Analytics

This project analyzes credit card customer behavior using **Python for data analysis** and **Tableau for interactive visualization**.  
The objective is to explore customer activity, understand spending patterns, identify churn risk, and present insights through a clean and interactive dashboard.

The analysis combines exploratory data analysis in Python with business-focused visualizations in Tableau to translate raw customer data into actionable insights.

---

## Tools and Technologies

- **Python** (Pandas, NumPy, Seaborn, Matplotlib)
- **Jupyter Notebook** for exploratory analysis
- **Tableau** for dashboard creation and visualization
- Data analysis and aggregation techniques for churn analysis

## Analytical Workflow

The project follows a typical data analysis pipeline:

1. **Data Exploration**  
   Initial inspection of customer demographics, credit limits, transaction counts, and account activity.

2. **Data Cleaning and Preparation**  
   Handling missing values, formatting variables, and preparing the dataset for analysis.

3. **Exploratory Data Analysis (EDA)**  
   Investigating customer spending behavior, transaction activity, and relationships between variables.

4. **Customer Behavior Analysis**  
   Identifying patterns related to customer engagement, transaction volume, and credit usage.

5. **Churn Analysis**  
   Evaluating how factors such as product engagement, spending behavior, and tenure relate to customer attrition.

6. **Dashboard Visualization**  
   Building an interactive Tableau dashboard to communicate key metrics and behavioral patterns.

---

## Key Insights Explored

- Overall customer churn rate and its distribution
- Differences in spending behavior between existing and attrited customers
- Impact of product engagement on customer retention
- Churn patterns across card categories and customer tenure

These insights can help organizations better understand customer engagement and identify potential strategies to improve retention.
